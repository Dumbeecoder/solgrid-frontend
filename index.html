Setup guide
Follow these steps exactly and you will be earning SOL in under 10 minutes. No experience needed — just copy and paste the commands below.
1
Create a free Solana wallet
You need a wallet to receive your SOL rewards. Phantom is the easiest — it's a free browser extension. Your wallet has an address (like a bank account number) and a private key (like your PIN — never share this with anyone).
1. Go to phantom.app and click Download
2. Install the browser extension
3. Click Create new wallet
4. Write down your 12-word recovery phrase on paper and keep it safe
5. Your wallet address looks like: 7xKXt... (a mix of letters and numbers)
6. You will need this address to register your node
Already have a Phantom wallet? Skip this step. You can use your existing wallet.
2
Install Go (needed to build the prover)
Go is a programming language the prover is built with. You only need to install it once — think of it like installing a driver for your printer.
# Install Homebrew first (a Mac package manager)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Then install Go
brew install go

# Confirm it worked
go version
Open Terminal: press Cmd+Space, type Terminal, press Enter. Then paste the commands above one at a time.
1. Go to https://go.dev/dl/
2. Download the Windows .msi installer
3. Run it and click Next through everything
4. Close and reopen PowerShell
5. Confirm it worked by typing:

3
Download and build the prover
The prover is the small app that runs on your computer and earns you SOL. We download it from GitHub and build it — takes about 2 minutes.
git clone https://github.com/Dumbeecoder/solgrid-prover
cd solgrid-prover
go build -o prover ./cmd/prover

You will see text scrolling for 1-2 minutes — that is normal. Wait until it returns to a blank line.
4
Run the prover and start earning
Now we start the prover with your wallet key. It connects to Solana's free public servers automatically — no extra setup needed.
# Replace YOUR_PRIVATE_KEY with your key from Phantom
# Phantom: Settings > Security and Privacy > Export Private Key

export PROVER_PRIVATE_KEY="YOUR_PRIVATE_KEY"
export NODE_RPC="https://api.mainnet-beta.solana.com"
export SOLGRID_API="https://api.solgrid.io/api"
export NODE_TYPE="sol-full"

./prover
🔒 Your private key never leaves your computer. It only signs messages to prove the node is yours.

# Create a system service so it runs automatically
sudo nano /etc/systemd/system/solgrid.service

# Paste this content:
[Unit]
Description=SolGrid Prover
After=network.target

[Service]
Environment=PROVER_PRIVATE_KEY=YOUR_KEY
Environment=NODE_RPC=https://api.mainnet-beta.solana.com
Environment=SOLGRID_API=https://api.solgrid.io/api
Environment=NODE_TYPE=sol-full
ExecStart=/root/solgrid-prover/prover
Restart=always

[Install]
WantedBy=multi-user.target

# Save and run:
sudo systemctl enable solgrid
sudo systemctl start solgrid
sudo systemctl status solgrid
