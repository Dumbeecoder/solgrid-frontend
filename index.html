How the reward pool works
Every week, a fixed SOL pool is distributed across all active nodes proportionally based on points earned. Points are earned by passing challenges every 5 minutes. The more nodes you run and the higher your uptime, the greater your share. Every distribution is executed by an on-chain Anchor program — no human controls the payouts, no trust required.

Setup guide
Follow these steps and you will be earning SOL in under 10 minutes. No experience needed — just copy and paste the commands below.
1
Create a free Solana wallet
You need a wallet to receive your SOL rewards. Phantom is the easiest — it is a free browser extension. Your wallet has an address like a bank account number and a private key like your PIN — never share your private key with anyone.
1. Go to phantom.app and click Download
2. Install the browser extension
3. Click Create new wallet
4. Write down your 12-word recovery phrase — keep it safe offline
5. Your wallet address looks like: 7xKXt... (letters and numbers)
6. You will need this address to register your node
💡 Already have Phantom? Skip this step and use your existing wallet.
2
Install Go
Go is the programming language the prover is built with. Install it once — like installing a printer driver.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install go
go version
💡 Open Terminal: press Cmd+Space, type Terminal, press Enter. Paste commands one at a time.
1. Go to https://go.dev/dl/
2. Download the Windows .msi installer
3. Run it and click Next through everything
4. Close and reopen PowerShell
5. Confirm it worked: go version
💡 Use PowerShell not Command Prompt. Press Windows key, type PowerShell, click it.
sudo apt update && sudo apt install -y golang-go git
go version
3
Download and build the prover
The prover is the app that earns you SOL. We download it from GitHub and build it — takes about 2 minutes.
git clone https://github.com/Dumbeecoder/solgrid-prover
cd solgrid-prover
go build -o prover ./cmd/prover

cd solgrid-prover
go build -o prover.exe ./cmd/prover

💡 Text will scroll for 1-2 minutes — that is normal. Wait for a blank prompt line.
4
Run the prover and start earning
Start the prover with your wallet private key. It connects to Solana's free public servers automatically.
# Get your key: Phantom > Settings > Security and Privacy > Export Private Key
export PROVER_PRIVATE_KEY="YOUR_PRIVATE_KEY"
export NODE_RPC="https://api.mainnet-beta.solana.com"
export SOLGRID_API="https://solgrid-api-production.up.railway.app/api"
export NODE_TYPE="sol-full"
./prover
🔒 Your key never leaves your computer. It only signs messages to prove the node is yours.

The prover must stay running to earn points. If you close the terminal it stops.

sudo nano /etc/systemd/system/solgrid.service
# Paste this:
[Unit]
Description=SolGrid Prover
After=network.target
[Service]
Environment=PROVER_PRIVATE_KEY=YOUR_KEY
Environment=NODE_RPC=https://api.mainnet-beta.solana.com
Environment=SOLGRID_API=https://solgrid-api-production.up.railway.app/api
Environment=NODE_TYPE=sol-full
ExecStart=/root/solgrid-prover/prover
Restart=always
[Install]
WantedBy=multi-user.target
# Save then run:
sudo systemctl enable solgrid
sudo systemctl start solgrid
